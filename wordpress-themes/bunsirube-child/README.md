# 文標 子テーマ

文標本体を直接編集せず、CSSや小さな関数追加で改造するための子テーマです。

## 入れ方

1. 親テーマ `文標` を先にインストール
2. `bunsirube-child-0.1.1.zip` または `bunsirube-child.zip` をWordPress管理画面からアップロード
3. `外観 > テーマ` で `文標 子テーマ` を有効化
4. CSSは `外観 > テーマファイルエディター > style.css`、小さな関数追加は `functions.php` に書く

## 使う理由

親テーマを直接編集すると、文標アップデート時に変更が上書きされます。
子テーマに改造を書けば、親テーマを更新しても改造内容を残せます。

## 向いている改造

- 余白、色、文字サイズの微調整
- CTA、ブログカード、比較表カードの見た目調整
- 小さなフィルターやショートコード追加

## 向いていない改造

- 親テーマ本体の大きな構造変更
- 外部プラグインとの衝突調査
- WordPress本体やサーバー設定の変更
