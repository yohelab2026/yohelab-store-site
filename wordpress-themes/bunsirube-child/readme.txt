文標を直接編集せず、CSSや小さな関数追加で改造するための子テーマです。

== Description ==

文標 子テーマは、親テーマ「文標」のアップデートを残しながら、見た目や小さな関数を安全に調整するためのテーマです。

== Installation ==

1. Install and keep the parent theme 文標.
2. Upload bunsirube-child-0.1.1.zip or bunsirube-child.zip.
3. Activate 文標 子テーマ.
