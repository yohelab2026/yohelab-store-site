Before you buy

買う前に分かるようにしたこと

無料テーマと比べて強いのは、デザインだけではなく「記事を書く前の型」が入っているところ。

誰向けか

ブログ初心者、アフィリエイト記事を書く人、小さなメディアを運営する人向け。企業サイト全体の大型テーマではない。

何が楽になるか

比較、レビュー、ランキング、FAQ、商品紹介など、記事の流れをパターンから選んで書き始められる。

何を保証しないか

検索順位、売上、AIの回答表示は保証しない。テーマは読み取りやすい構造と運用の手間を減らすための土台。

Install guide

WordPressに入れる手順

迷わないように、管理画面で触る順番を固定している。

1. ZIPをアップロード

外観 → テーマ → 新規追加 → テーマのアップロードから、ZIPを選ぶ。

2. テーマを有効化

テーマを有効化したあと、外観 → よへラボ記事テーマ で設定画面を開く。

3. 色と解析を設定

メインカラー、文字色、GA4、Search Console、内部解析、llms.txtを同じ画面で設定する。

4. 記事テンプレートを使う

投稿編集画面の「パターン」から、比較、レビュー、案件紹介、FAQ記事、ランキング、商品紹介を選ぶ。

5. 解析を見る

外観 → よへラボ記事テーマ の解析欄で、PV、人気記事、流入元を確認する。

6. AI向け案内ファイルを確認

サイトURLの末尾に /llms.txt を付けて、案内ファイルが出るか確認する。

Plugins

入れていいプラグイン、減らせるプラグイン

全部をテーマに詰め込むのではなく、テーマで代替できるものと、専門プラグインに任せるものを分ける。

入れていい

バックアップ、セキュリティ、問い合わせフォーム、EC/決済、画像最適化、スパム対策。サイト全体や外部連携に関わるものは専用プラグインでよい。

減らせる

FAQ構造化データ、目次、GA4タグ挿入、Search Console認証、llms.txt、簡単なアクセス解析はテーマ側に入れている。プラグインを増やしすぎない設計。

重複注意

SEO、キャッシュ、解析は重ねるほど良いわけではない。同じ役割のプラグインを複数入れない。
